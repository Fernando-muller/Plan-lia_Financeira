# 🎯 Objetivo do Sistema

Desenvolver um sistema desktop para controle financeiro pessoal, permitindo ao usuário gerenciar suas entradas e saídas de dinheiro de forma simples e eficiente.

# ⚙️ Funcionalidades do Sistema
🔐 Tela de login
💰 Cadastro de movimentações (Entrada e Saída)
📊 Cálculo automático de saldo
📋 Exibição de extrato financeiro
💾 Integração com banco de dados MySQL
🔄 Atualização em tempo real das informações
